# Haris Laeeq — Portfolio

This is my personal portfolio website built using Next.js 14.  
It showcases my projects, skills, and background in a clean, animated UI.

The goal of this project was to create a fast, modern, and responsive portfolio that reflects real-world frontend and full-stack development practices.

---

## 🚀 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- CSS Modules
- Google Fonts (Syne + DM Sans)
